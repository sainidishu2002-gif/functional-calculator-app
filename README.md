# Functional Calculator App

Functional Calculator App offers a colorful, responsive web interface with Simple Interest, BMI, and Calorie calculators. Instantly compute finance and health metrics using modern HTML, CSS, and JavaScript. Easy to use, perfect for students and daily needs!

## Calculators Included

- **Simple Interest Calculator**  
  Enter principal, rate, and time to calculate simple interest.

- **BMI Calculator**  
  Input your height and weight to determine your Body Mass Index.

- **Calorie Calculator**  
  Estimate daily caloric needs based on your age, gender, height, weight, and activity level.
